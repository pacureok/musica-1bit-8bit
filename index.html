Música Arcade: 1-bit & 8-bit

Crea sonidos retro como los de los viejos arcades. Controla la frecuencia y el volumen para personalizar tus tonos de "1" y "0".

Descargar Audio

Cómo funciona:

  • Usa los deslizadores de Frecuencia Base y Volumen para ajustar el sonido general.
  • El botón '1' reproduce un tono ligeramente más alto.
  • El botón '0' reproduce un tono ligeramente más bajo.
  • Mantén pulsados los botones '1' o '0' para escuchar el tono, suéltalos para detener.
  • 'Detener Sonido' apaga cualquier sonido activo o secuencia.
  • También puedes usar las teclas '1' y '0' de tu teclado para activar los sonidos.
  • En "Ingresar Secuencia", puedes escribir parámetros separados por comas.
    • Para un solo tono: [tipo], [frecuencia en Hz], [volumen en %] (ej: 1, 300hz, 80%).
    • Puedes omitir 'tipo' si solo quieres un tono específico (ej: 300hz, 80%).
    • Para una secuencia de '1' y '0': 1,0,1,0.
    • Puedes combinar: 1, 300hz, 80%, 0, 150hz, 50% para tonos específicos en secuencia.
  • Usa "Iniciar Grabación" antes de generar el sonido y "Detener Grabación" para guardar el archivo WAV.
